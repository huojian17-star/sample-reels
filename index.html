艾瑞斯大陆设定集

独立构建的原创奇幻世界观 · 核心机制与世界全景[cite: 2]

「艾瑞斯大陆」是一个以"物理侧""魔法侧""研究侧"三足鼎立为核心的奇幻世界[cite: 2]。物理侧者锤炼肉身与兵器[cite: 2]，魔法侧者沟通元素与秘法[cite: 2]，而无能力的研究侧则依靠知识与制度居中制衡——三者的权力博弈构成了整个世界的底层张力[cite: 2]。

能力体系

  • 物理侧：剑术师 · 骑士 · 刺客 · 狂战士 · 盾兵 · 枪兵 · 弓兵[cite: 2]
  • 魔法侧：元素法师 · 咒术师 · 幻术师 · 治愈师 · 屏障建构师 · 魔导师 · 召唤师[cite: 2]
  • 研究侧：无能力，主导公会系统与社会治理[cite: 2]
  • 职阶晋升：主修+辅修双职阶制 · 跨侧能量冲突会导致暴毙警告[cite: 2]

五大城市

  • 寰宇市：物理侧聚集区，尚武粗犷（苏式中期风格）[cite: 2]
  • 天际市：魔法侧聚集区，神秘高雅（索莱恩秘法传承）[cite: 2]
  • 中心市：研究侧总部，秩序理性（中美体制混合物）[cite: 2]
  • 岩湖市：矿业重镇，寡头竞争（穹顶矿脉垄断）[cite: 2]
  • 富华城：商业中心，市井浮华（六十年代美国风格）[cite: 2]

社会系统

  • 货币体系：金狮鹫币—银鹿角币—棕熊铜币三级汇兑[cite: 2]
  • 政治架构：研究侧主导行政[cite: 2]，元元老会议最高决策[cite: 2]
  • 教育体系：6-12岁义务教育 → 能力者公会考试 / 普通升学双轨[cite: 2]
  • 法律体系：《艾瑞斯大陆宪法》+《能力者管理总则》等六部法典[cite: 2]

核心历史

  • 奥莱因战争：神之子「尤尔」统合物理与魔法联军封印暗界通道[cite: 2]
  • 秘法之断：战后物理/魔法侧决裂，奥莱因王国陨落[cite: 2]
  • 元老会议历：以会议成立之年为起点，12月×30日+5庆典日[cite: 2]
  • 穹顶矿脉：稀有能量晶体的发现重塑了大陆力量格局[cite: 2]

暗界与机密

  • 暗界五大假说：失败创世 / 古神尸骸 / 实验失控 / 异维度入侵 / 集体意识阴影[cite: 2]
  • 玻利瓦尔档案：一级机密，魔法侧"源头"的真实本质存疑[cite: 2]
  • 神话位阶：十阶「法则化身」，仅尤尔、玻利瓦尔、调停者三位历史案例[cite: 2]

经济、艺术与连载

  • 维尔经济学双著：《银行、货币体系与经济命脉》《论能力者与非能力者的劳动》[cite: 2]
  • 四大艺术流派：奥莱因现实主义 · 索莱恩秘义派 · 结构主义 · 世俗派[cite: 2]
  • 吟游连载精选：包含正在筹备扩展的原创故事《渡鸦镇》[cite: 2]
