鸦林镇疑案

Crowwood Mystery · 1989

A pixel-art detective text adventure set in a small town where nothing ever happens — until it does.

Features

线索笔记 (N)
每条线索自动记录。注意，有些线索可能是误导信息。
思维笔记 (T)
自由输入推理，连接关键词解锁隐藏推论。关键节点必须整理思路才能推进。
动态审讯
审讯选项取决于你实际收集到的线索。选错线索会被嫌疑人回怼。
多重结局
完美定罪、部分正义、无罪释放、抓错人 — 取决于调查质量和关键选择。
