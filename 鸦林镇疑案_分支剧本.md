# 悬疑文字解谜游戏：鸦林镇疑案
## 分支剧本 v1

---

## 一、序幕（固定线）

*1989年11月，美国东部，鸦林镇。*
*你，42岁的警长，在这个只有三千人的小镇已经干了十五年。*
*这里鲜有大事——上一次命案还是十二年前，对，就是那对死在自家客厅的老夫妻。*
*直到今天下午四点零三分。*

接线员：警长，玫瑰街114号的报警电话。女性，语气慌张，说邻居躺客厅地上一动不动。

你选择如何回应？（仅影响开场对话，不改变主线）

**分支 1A**："我马上到，告诉她别进去、别碰任何东西。"
- 接线员传达指令，女报案人在家门口等。现场未被干扰。

**分支 1B**："问她邻居还有没有呼吸。"
- 女报案人战战兢兢推门进去确认，在客厅地毯上踩了咖啡渍，留下半个脚印。这个脚印在后面审讯时可能误导你指向错误嫌疑人。
- *隐藏后果：你会在"厨房vs客厅"证据链条中多一个干扰项*

---

## 二、现场勘查（调查路线分支）

你带四人到了玫瑰街114号。警戒线拉好。

你面前是一栋典型的二层独栋住宅，奶油色外墙，门前草坪因为秋冬有些枯黄。车库的门半开着。

这时你需要决定从哪里开始搜查。（选择顺序，不是二选一）

### 调查路线 A：先搜查室内 → 再搜车库
- 你先进客厅。女主人（约35岁，浅棕色短发，穿着围裙）面朝下倒在沙发和茶几之间。地毯上是大片深褐色咖啡渍，已经干了。碎瓷片——应该是马克杯——散在她身体旁边。
- 厨房餐桌上两块三明治。一块完整，一块咬了两口。旁边两杯橙汁，都没喝过。
- 二楼第一间儿童房：少年躺在床上，被子盖到胸口，你差点以为他睡着了。但他不呼吸了。脖子上有轻微按压痕迹。枕头在地板上。
- 主卧：唯一让你觉得不对劲的，是床头柜上那个相框——一家三口的合影被仔细擦拭过，没有灰尘。

然后你去车库。工具箱少了一把锤子。皮卡不见了。轮胎印延伸出车道，往西。

### 调查路线 B：先搜车库 → 再搜室内
- 你先看车库。皮卡不在。工具箱缺了一把锤子——挂孔的位置非常明显，其他工具都整整齐齐。地面上有不明显的新鲜摩擦痕迹，轮胎印还很清晰，泥土是湿的（昨天下午下过小雨）。
- 然后你进客厅。因为已经知道皮卡去向，你下意识注意到了地毯边缘有一个不显眼的靴印——不是女主人的拖鞋造成的。这个靴印的泥土，和车库的泥土颜色一致。
- *隐藏收获：先搜车库会让你注意到泥土线索，从而在后续审讯中更快识破工友的谎言*

## 三、水库发现（固定推进）

底下人跑来汇报：三里外的鸦湖水库，皮卡沉在水里。男主人坐在驾驶位上，头靠在方向盘上。没有外伤。而锤子不在车上。

法医初步判断：女主人后脑创伤，约昨日下午五点死亡。儿子疑似窒息，时间接近。男主人……看上去像是自己开进去的。

你站在这辆刚从水里拖出来的皮卡旁，雨停了，冷风在湖面上吹。直觉告诉你这不是自杀。但证据还差得远。

接下来，你要分配三个手下的任务——每个任务都会得到不同线索，但**你只有两个手下可以在一开始就派出去**，这意味着有一个情报会被延迟。

**选择两者、遗弃其一**：

| 选择 | 即时获得 | 延迟失去 |
|---|---|---|
| 调查男主人社会关系 | 伐木场工友的名字 | 家政人员身份会晚半天才发现 |
| 调查女主人的过去 | 前夫的存在 + 酒吧线索 | 工友的安眠药线索晚半天 |
| 调查儿子的学校生活 | 霸凌事件 + 家政董事长 | 水库死亡时间分析晚半天 |

*延迟的线索不会永远失去——你会在后续审讯中意外获得，但可能来不及阻止某些事情发生（比如工友可能会逃跑）。*

## 四、审讯分支（双线影响）

### 审讯一：前夫

前夫是个瘦高男人，三十多岁，眼睛旁边有一道旧伤疤。他坐在审讯室里，手指一直敲桌子。

你面对他有三种态度（选一种）：

**态度 A：冷静盘问**
- "把你和这家人最后一次见面说清楚。"
- 前夫不会一开始就提他是少年生父的事。你需要在对话中抓住他话里的漏洞，才会触发他说出"其实他是我儿子"的坦白。
- *效果：获取"前夫认子"线，但他之后不会再主动提供更多信息。*

**态度 B：施加压力**
- "我们有目击者看见你在附近。你知道我们迟早会发现。"
- 前夫被激怒，情绪失控。他会骂女主人，但也因为愤怒而泄露了"她以前和一个公司高层有过一腿"的线索。
- *效果：提前获取"董事长+女主人一夜情"的线索，但他否认自己是生父，你必须在后续拿到DNA报告才能反驳他。*

**态度 C：骗取信任**
- "我们不是来抓你的。你是受害者，对吧？你有话想说很久了。"
- 前夫先沉默，然后崩溃。他会主动交代自己和少年不是父子关系（他其实知道但不敢面对），并且会提供酒吧不在场证明的所有细节。
- *效果：一次性获得最多信息，但需要高"审讯技巧"值——实际上，前夫是个容易心软的人，只要你不刺激他，他什么都说。*

---

### 关键分歧点：DNA 报告出炉

无论你之前怎么审前夫，DNA 报告会在此刻送达。

报告说：少年的 DNA 和前夫不匹配。也和男主人不匹配。

此时你有两个方向：

**走向 1：再传前夫**
- 前夫二次提审。如果你之前态度是 A 或 B，他会情绪崩溃并透露出"她还有一个男人，某个公司高层"。如果你之前选了 C，他会补充更多：他曾经在酒吧偶然看见那个高层和女主人说话，对方开的车上有"金斯顿家政"的标志。
- *作用：明确董事长嫌疑*

**走向 2：直接调查家政公司**
- 跳过前夫，直接传唤董事长。但他有律师，他会冷笑说"根据宪法第五修正案，我有权保持沉默"。你什么也问不出。
- *坏结局风险：如果以后缺乏其他证据，董事长会成功逃脱*

---

### 审讯二：董事长

董事长是个五十岁左右的男人，头发灰白，西装笔挺。即使坐在审讯室里，他看起来也像是在会议室。他的律师坐在旁边，表情比他还紧张。

三种方式中你选一种：

**方式 A：直接施压**
- "事发当天你没去董事会。你去哪了？"
- 律师代答："我的委托人有权不回答。"
- *结果：零信息。但如果你此前得到了前夫提供的"金斯顿家政"线索，你可以问："十二年前你和女主人有过什么？"——董事长会沉默。律师会要求暂停审讯。你至少知道他怕这件事。*

**方式 B：商业调查**
- 你让两名警员去查家政公司的财务记录和车辆调度。同时你在审讯室里和董事长闲聊。
- 他会放松警惕，聊起自己的公司、竞争对手、以及"某些不听话的员工"。
- *结果：不会直接得到命案信息，但你会发现——他的公司只有董事长本人和两个高级经理有权签发工作服。其中两个经理当天都在公司，那么穿着家政服装出现在门口的人只能是……*

**方式 C：找到儿子留的线索**
- 如果你派人去加拿大找到了他儿子（通过国际刑警协助），可以拿到一份证词——儿子承认自己在学校被霸凌，也承认他告诉了父亲，并且那天晚上父亲慌乱地安排了转学。
- *结果：董事长防线崩溃，开始交代部分事实。但他会把罪责全推到工友身上。*

---

### 工友失联 & 木屋发现

工友的尸体在距离水库 1 公里的废弃木屋被发现。同样安眠药过量。同样无外伤。

此时如果你**之前在"选择两者遗弃其一"中选了调查女主人过去**，你的手下会在木屋发现一个关键物证：一张揉皱的纸条，上面用铅笔写着——"明天下午四点，疯牛酒吧后巷，钱和你的东西都在。"

如果你**选了调查男主人社会关系**，你会在木屋提前发现这纸条，并立刻辨认笔迹——不是工友的字。

如果两个都没选（选了儿子学校），那么纸条会被雨水泡烂，无法辨认。

## 五、收网（最终分支）

到了这一步，你掌握的证据可能是：

- DNA 报告（必备）
- 锤子位置及指纹/血迹（必备，除非你忘了搜下水道）
- 纸条笔迹鉴定（分支条件）
- 加拿大儿子的证词（分支条件）
- 家政公司员工证词（分支条件）
- 安眠药购买人证词（分支条件）
- 邻居/目击者证词（必备）

### 结局 A：完整收网（完美结局）
*条件：纸条 + 至少两个分支证据 + 锤子*
- 你完整还原案发过程。董事长、工友（已死）的罪行全部公开。
- 前夫虽然暴怒但也被保护起来（他知道了少年不是自己亲生，但也接受了）。
- 小镇报纸头版报道。最后一幕是你站在警察局门口，看着街上的人。天色渐暗。字幕："有些杀人案不需要凶器。安眠药就够了。"

### 结局 B：部分定罪（普通结局）
*条件：锤子 + 至少一个分支证据，但纸条缺失*
- 你有足够证据给董事长定罪（安眠药 + 家政服装 + 儿子证词）。但工友的死亡被定为"畏罪自杀"——你其实知道不是自杀，但没有证据证明董事长直接杀了工友。
- 董事长被判终身监禁。你在法庭外看见他的儿子——那个12岁的转学生，现在完全不知道自己的父亲做了什么。他看了你一眼，上了飞往加拿大的飞机。

### 结局 C：有罪未罚（悲剧结局）
*条件：锤子证据缺失 + 公司调查不完整*
- 陪审团认为证据不足。董事长无罪释放。他去加拿大和儿子团聚了。
- 你在办公室看着结案报告。咖啡冷了。外面又下雨了。
- 字幕："在鸦林镇，有些事不会上报纸。"

### 结局 D：怪错了人（隐藏结局）
*条件：你所有审讯中都选了"施加压力"风格 + 锤子证据你从未找到*
- 前夫被定罪。工友的安眠药被判定为"男主人自杀的药物来源"。凶手逍遥法外。
- 三个月后，你收到一封从加拿大寄来的匿名信。没有署名。信纸是家政公司的抬头。上面只有一句话：
- "警长，其实那天你离真相很近了。差一点。"
- 你把信纸揉成团，扔进废纸篓。但你又捡起来，放进了抽屉。

## 七、技术实现要点（Godot）

### 场景结构
- `main.gd`：全局管理器
- `scenes/investigation/`：六个调查场景（客厅/厨房/卧室/车库/水库/伐木场）
- `scenes/interrogation/`：审讯室（复用），四个嫌疑人
- `scenes/caseboard/`：线索面板（玩家整理的物证+口供）
- `scenes/ending/`：结局分支

### 分支追踪
用一个全局字典追踪所有决策点：
```gdscript
var case_state = {
  "investigation_order": "",      # "indoor_first" / "garage_first"
  "first_agents": [],              # 最先派出的任务
  "husband_style": "",             # "calm" / "pressure" / "trust"
  "dna_path": "",                  # "reinterrogate" / "investigate"
  "chairman_style": "",            # "direct" / "business" / "canada"
  "found_hidden_clue": false,      # 纸条
  "found_hammer": false,           # 锤子
  "hammer_fingerprints": false,    # 锤子指纹
  "evidence_quality": 0,           # 0-100 分数
}
```

### 线索矛盾标记系统
玩家可以在任意两件证据之间"拉线"标记矛盾，比如：
- "前夫口供：孩子是我的" ↔ "DNA报告：不匹配"
- "董事长口供：我有权保持沉默" ↔ "家政员工证词：他当天缺席董事会"

每个矛盾被正确标记后，`evidence_quality` +20 分。

## 八、开发优先级

**Demo 版（先做这个）**：
1. 序幕 + 现场勘查（2 个场景，分支 A/B）
2. 审讯前夫（3 种态度）
3. DNA 反转
4. 一个结局（完美结局）
5. 像素黑白美术 + 打字机效果

Demo 可以独立体验约 20-25 分钟。

**完整版**需要新增：
- 董事长审讯
- 工友木屋发现
- 全部四个结局
- 矛盾标记系统
- 线索面板
