经济系统数值模型

以下是基于世界观货币体系（1金狮鹫 = 10银鹿角 = 500棕熊铜币）设计的怪物掉落与任务奖励数值逻辑。

怪物掉落金币公式

DropGold = MonsterLevel × 3 + Random(-2, 5) + Floor(Luck ÷ 10)

其中 MonsterLevel 为怪物等级（1-20），Random(a,b) 为区间随机整数， Luck 为角色幸运值（0-100）。掉落货币单位为棕熊铜币。
示例：Lv.5 影犬 + 幸运值 35 → DropGold = 5×3 + Random(-2,5) + Floor(35÷10) = 15 + [−2~5] + 3 = 16~23 棕熊铜币。

幸运值对稀有掉落的影响

RareDropRate = BaseRate × (1 + Luck ÷ 200)

BaseRate 为基础稀有掉落率（通常3%-8%），Luck 为角色幸运值。
示例：BaseRate = 5%，Luck = 80 → RareDropRate = 5% × (1 + 80/200) = 5% × 1.4 = 7%。 这使得幸运值从0到100的收益曲线为线性递增，不会出现边际递减——保证玩家对幸运属性的持续追求。

公会任务报酬定价模型

TaskReward = BaseReward × (1 + RiskMultiplier) × (1 - GuildCut)

其中 BaseReward = 任务基础报酬（由威胁等级、时长、团队规模核定），
RiskMultiplier = 风险系数（影犬清剿 ≈ 0.2，深渊领主讨伐 ≈ 2.5），
GuildCut = 公会抽成（10%-20%，用于抚恤基金与运营）。
示例：清剿影犬 BaseReward = 40银鹿角，RiskMultiplier = 0.2，GuildCut = 0.15
→ TaskReward = 40 × 1.2 × 0.85 = 40.8 银鹿角（实际发放取整为41银鹿角）。

三角色职业定位设计

基于艾瑞斯大陆的双侧体系，设计一套三人小队协作玩法。每个角色拥有独特的职业定位与技能差异，形成"侦察→突破→支援"的战术闭环。

观测者

侦察 · 情报 · 控制
  • 定位：魔法侧幻术师/咒术师
  • 核心能力：远程感知敌人位置、揭示隐藏通道、制造幻象迷惑敌人
  • 战斗风格：不直接参与近战，通过"标记"为队友提供伤害加成。可对敌人施加减速/混乱状态
  • 设计意图：满足策略型玩家的信息优势需求。低血量、低防御，但技能决定团队的战术上限

突破者

近战 · 伤害 · 坦克
  • 定位：物理侧剑术师/狂战士
  • 核心能力：正面突破、吸引仇恨、高爆发单体伤害
  • 战斗风格：站在队伍最前方，承受主要伤害。连击系统（3段→5段）随等级解锁。怒气值满后可释放"破甲突刺"（三阶技能）
  • 设计意图：满足动作型玩家的爽感需求。操作门槛低但上限高——高手可通过精准格挡时机实现无伤通关

策应者

道具 · 治疗 · 辅助
  • 定位：魔法侧治愈师 + 研究侧工程师
  • 核心能力：治疗队友、投掷道具（烟雾弹/陷阱/增益药剂）、环境互动（修复桥梁/破解机关）
  • 战斗风格：不直接攻击，但通过道具组合可以改变战场地形。例如：投掷油瓶 + 观测者的火系标记 = 范围燃烧
  • 设计意图：满足辅助型玩家的创造力需求。表面上是"混子位"，实际上道具连携是Boss战的核心解法

注：该三角色设计遵循"猜拳式克制 + 连携增益"的原则。观测者标记敌人 → 突破者享受伤害加成 → 策应者利用突破者创造的输出窗口投掷关键道具。三人同时行动时触发"羁绊技"，具体效果取决于小队编成中的侧系组合。
